# Example Bundle

This directory is a non-sensitive example of the Gravity proof bundle structure.

It is included to show reviewers and technical users the expected layout of a certification package.

Example files may include:

- `artifact.json`
- `analysis.json`
- `hash_manifest.txt`
- `verification_report.txt`

These example files are for format demonstration only.
