# GravityBTC Standards

This directory contains published GravityBTC standards and their verification artifacts.

## Structure

- `*.html` — Released public standard documents.
- `hashes/` — SHA-256 hashes for released standards.
- `records/` — Verification records for released standards.

## Published Standards

- Gravity Evidence Preservation Standard v1.0 (GEPS-1)
- Gravity Publication Integrity Standard v1.0 (GPIS-1)

These standards are published by GravityBTC under the Gravity Standards Authority framework.

Canonical public versions are available at:

- https://gravitybtc.space/standards/gravity-evidence-preservation-standard-v1.html
- https://gravitybtc.space/standards/gravity-publication-integrity-standard-v1.html
