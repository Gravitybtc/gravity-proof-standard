# GravityBTC Publication Model

GravityBTC maintains public materials through two coordinated publication surfaces.

## Canonical Website

The canonical public reading surface is:

https://gravitybtc.space/

The website presents standards, articles, verification records, publication references, and retrieval materials in their public operational context.

## GitHub Repository

This repository provides a public, version-controlled mirror of selected GravityBTC standards, articles, hashes, verification records, and publication references.

GitHub is used for transparency, source control, public review, and long-term reference.

## Directory Model

- `standards/` — Released GravityBTC standards and their verification artifacts.
- `articles/` — Published research and explanatory articles.
- `article-hashes/` — SHA-256 hashes for published articles.
- `publication-references/` — Public publication reference records.
- `verification-records/` — JSON verification records.
- `example-bundle/` — Example verification bundle material.

## Authority Rule

The website is the canonical public reading surface.

GitHub is the version-controlled public mirror.

Released materials should not be edited silently. Updates should be versioned, hashed, recorded, and published through the GravityBTC publication workflow.
